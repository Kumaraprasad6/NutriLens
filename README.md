# NutriLens

A SwiftUI food tracking app for Indian cuisine, built with Clean Architecture.

**Status:** MVP in development. Active work on `develop` branch.

## Tech Stack
- SwiftUI + SwiftData (iOS 17+)
- Clean Architecture (Data / Domain / Presentation)
- MVVM with `@Observable`
- Protocol-based Dependency Injection

## Branches
- `main` — Stable releases (currently empty until MVP)
- `develop` — Active development branch
- `feature/*` — Individual features
